How to install

  1. Tap Download for Android above and open the downloaded .apk file.
  2. If Android warns about an unknown source, tap Settings → enable Allow from this source, then go back.
  3. Tap Install, then open Tadapop from your app drawer.

Good to know

  • Android only. This is an .apk — it can't be installed on an iPhone.
  • Solo tracking needs no account. An account (a quick email code) is only needed for the Arena.
  • This is a pre-release test build, so expect rough edges — that's what we're testing.
